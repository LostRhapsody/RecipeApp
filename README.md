# Evan's Nuxt Quick Start Template

A simple Nuxt app that's been pre-installed with:

- Tailwind CSS 4
- Oxfmt
- Oxlint
- Nuxt UI
- Better Auth
- Nuxt SEO
- Cloudflare Turnstile
- Zod

Just the essentials.

There is no sqlite adapter for better auth, yet.

# Components

- Form
  - Nuxt UI
  - Turnstile
  - Zod
  - Great for an example
  - Doesn't send an email, it's just an example.
